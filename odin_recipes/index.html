将github项目与本机电脑相连

  1. 打开github项目
  2. 点击Code按钮
  3. 复制项目链接
  4. 打开git bash
  5. 输入命令：git clone 复制的项目链接
  6. 等待项目下载完成
  7. 打开项目文件夹
  8. 在git bash中输入命令：git status
  9. 如果显示：On branch master Your branch is up to date with 'origin/master'. 说明已经连接成功
  10. 如果显示：fatal: Not a git repository (or any of the parent directories): .git 说明还没有连接成功，需要先进行连接
  11. 在git bash中输入命令：git remote add origin 复制的项目链接
  12. 在git bash中输入命令：git push -u origin master
  13. 等待项目上传完成
  14. 打开本机电脑的github项目，点击Pull requests按钮，点击New pull request按钮，选择master分支，点击Create pull request按钮，等待项目合并完成
  15. 恭喜你，你已经成功将github项目与本机电脑相连
